
Istruzioni per l'installazione e la compilazione del progetto Book Recommender

Requisiti di sistema

Java JDK 11 o superiore
Apache Maven 3.6.3 o superiore
PostgreSQL 12 o superiore
2GB di RAM minimo
500MB di spazio disco libero

Preparazione dell'ambiente

1. Installazione di Java
Assicurarsi di avere Java JDK 11+ installato. Per verificare, eseguire in terminale:

bashjava -version

Se Java non è installato o la versione è inferiore alla 11, installarlo da: https://adoptium.net/

2. Installazione di Maven
Verificare che Maven sia installato:
bashmvn -version
Se Maven non è installato, seguire le istruzioni su: https://maven.apache.org/install.html
